# Comparative Analysis of Transfer Learning Strategies on Caltech-101 Using Pre-Trained CNN Architectures

In this study, an end-to-end deep learning pipeline was deployed, comprising dataset curation, data preprocessing, pre-trained model loading, feature extraction, fine-tuning, and ulti-mate evaluation. One of our primary objectives was to compare the performance of models with frozen and unfrozen (fine-tuned) layers. Such comparisons help to explain the difference between these two in terms of training time and accuracy.

We have two files: A pdf file of the report and ipynb of the code used, The code in the ipynb is built in such a way that it is reproducible, given the dataset used is the exact same, dataset link: https://data.caltech.edu/records/mzrjq-6wc02
